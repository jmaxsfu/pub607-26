# Homework: Editing text with markdown and pandoc

1) Have a look at the "OYO-PandocTutorial" file, and try out the examples. 

2) With the content you've chosen to work on, try converting it to HTML and view it in your web browser.

   You'll probably want to do some editing to get to a clean, tidy, markdown file that does exactly what you want it to do.

3) I've included here a sample CSS stylesheet for you to play with. Make a copy of this file so that it's in the same folder as your own files above (you can just "Save as" in Sublime Text) .We'll talk about CSS a lot more over the next few sessions, but you can use this one as an example when you convert to HTML. I made this terrifically ugly on purpose: to provoke you into making changes!

4) Try doing some global search-and-replace on your Essay. Can you change all of your headings to another level? E.g., make all your first-level headings into second-second level headings, or vice-versa.

   And, have a look at "OYO-RegExTutorial" -- there are some challenge exercises at the end of that file.
