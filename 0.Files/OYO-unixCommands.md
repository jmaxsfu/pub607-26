## Some Unix Commands'


`pwd` -- print working directory (where am I?)

`cd *folder*` -- change directory

`cd *folder/subfolder*` -- (Mac) change to subfolder 

`cd *folder\subfolder*` -- (Windows) change to subfolder 

`cd ..` -- change directory to the parent folder  (go 'up')

`ls` -- list files (more usefully as ls -l 'long-form')

`more` -- show file one screenful at a time

`cat` -- concatenate (reel off these files to standard output) 

`split` -- split a file into multiple files

`head` -- show first part of a file

`tail` -- last part
 
`wc` -- word count

`cp *file* *folder*` -- copy a file into a folder

`mv *file* *folder*` -- move a file into an existing folder

`mv *file* *newfilename*` -- change name of file

`rm *file*` -- remove (delete) file

`grep` -- find strings or patterns of text in files

`sed` -- batch search and replace tool

`man *command name*` -- manual for any Unix command

`|` -- pipe output from one program into another program

`>` -- redirect output to a new file

`>>` -- redirect output to the end of an existing file

standard output (STDOUT): by default, your screen. But redirectable.

standard input (STDIN): by default, your keyboard. But redirectable.
